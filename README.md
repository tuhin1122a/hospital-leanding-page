# নূরজাহান হসপিটাল ম্যানেজমেন্ট সিস্টেম (Nurjahan Hospital Management System) - ফ্রন্টএন্ড

এটি নূরজাহান হসপিটাল ম্যানেজমেন্ট সিস্টেমের ফ্রন্টএন্ড রিপোজিটরি। আধুনিক ওয়েব টেকনোলজি ব্যবহার করে অ্যাডমিন, ডাক্তার, স্টাফ এবং রোগীদের জন্য একটি অত্যন্ত সুন্দর, সিকিউর এবং রেস্পন্সিভ ড্যাশবোর্ড ডেভেলপ করা হয়েছে।

## 🚀 টেকনোলজি স্ট্যাক (Tech Stack)

- **ফ্রেমওয়ার্ক**: [Next.js 16](https://nextjs.org/) (অ্যাপ ডিরেক্টরি)
- **কোর লাইব্রেরি**: [React 19](https://react.dev/)
- **ডিজাইন ও স্টাইলিং**: [Tailwind CSS v4](https://tailwindcss.com/)
- **UI কম্পোনেন্টস**: [Radix UI](https://www.radix-ui.com/) (অ্যাক্সেসিবল হেডলেস কম্পোনেন্ট) এবং [Framer Motion](https://www.framer.com/motion/) (সুন্দর অ্যানিমেশন এর জন্য)
- **ডেটা ফেচিং ও ক্যাশিং**: [@tanstack/react-query](https://tanstack.com/query/latest)
- **ফর্ম হ্যান্ডেলিং**: [React Hook Form](https://react-hook-form.com/) এর সাথে [Zod](https://zod.dev/) ভ্যালিডেশন
- **রিয়েল-টাইম ইঞ্জিন**: Socket.io-client (লাইভ ইভেন্টস ও চ্যাটিং এর জন্য)

## ✨ বিস্তারিত ফিচারসমূহ (Details Features)

1. **রোল-ভিত্তিক সিস্টেম (Role-Based Dashboard)**:
   - অ্যাডমিন, ডাক্তার, রোগী এবং স্টাফ—প্রত্যেকের জন্য আলাদা আলাদা কাস্টমাইজড ড্যাশবোর্ড।
   - শুধুমাত্র অনুমোদিত ইউজাররাই নির্দিষ্ট পেজগুলোতে ভিজিট করতে পারবে। সিকিউর কুকি (Secure HTTP-Only Cookies) ব্যবহার করে সেশন ম্যানেজমেন্ট করা হয়েছে।

2. **পেশেন্ট পোর্টাল এবং সেল্ফ-বুকিং**:
   - রোগীদের জন্য আলাদা একটি শক্তিশালী পোর্টাল।
   - রোগীরা নিজেরা রেজিস্ট্রেশন করে একাউন্ট তৈরি করতে পারবে এবং সরাসরি তাদের পছন্দের ডাক্তারের অ্যাপয়েন্টমেন্ট বুক (Self-Book) করতে পারবে।
   - পূর্বের সকল বুকিংস, প্রেসক্রিপশন ও অ্যাপয়েন্টমেন্ট স্ট্যাটাস নিজের প্রোফাইল থেকে দেখা যাবে।

3. **প্রিন্ট-রেডি ডকুমেন্টস**:
   - এডমিশন, ডিসচার্জ ও বিলিং এর জন্য সুন্দর ইনভয়েস জেনারেট এবং সরাসরি প্রিন্ট করার অপশন।
   - অ্যাপয়েন্টমেন্ট কনফার্মেশনের জন্য কাস্টম "অ্যাপয়েন্টমেন্ট টিকেট" (Printable Tickets) জেনারেট করা যায়।

4. **ডার্ক মোড এবং রেস্পন্সিভ ডিজাইন (Premium UI)**:
   - ব্যবহারকারীর চোখের স্বস্তির জন্য সিস্টেমটিতে ডার্ক এবং লাইট মোড টগল (Dark/Light Mode Setup) সুবিধা রয়েছে।
   - মোবাইল, ট্যাবলেট বা ডেস্কটপ—সব ধরনের ডিভাইসে ওয়েবসাইটটি একদম পারফেক্টলি মানিয়ে নিতে পারে (Fully Responsive)। মোবাইল ইউজারদের জন্য সাইড-বার স্লাইড ইন (Slide-in) মেন্যু রাখা হয়েছে।

5. **অ্যাডমিনিস্ট্রেটিভ কন্ট্রোল**:
   - স্টাফ ম্যানেজমেন্ট, স্টাফদের স্যালারি তৈরি হিসাব-নিকাশ (Salary features)।
   - ডাক্তারদের শিডিউল, পেশেন্ট এডমিশন, বিলিং ও এপ্রুভাল (Approvals) ম্যানেজমেন্ট।
   - সিস্টেমের সার্বিক ডেটা নিয়ে চার্টস এবং স্ট্যাটিস্টিকস (Analytics Dashboard)।

6. **রিয়েল-টাইম নোটিফিকেশনস ও চ্যাট**:
   - সকেট কানেকশন ব্যবহার করে যেকোনো নতুন অ্যাপয়েন্টমেন্ট বা নোটিফিকেশন সাথে সাথে ড্যাশবোর্ডে শো করবে, কোন পেজ রিলোড করা ছাড়া।

## 📦 প্রজেক্টটি লোকালি চালানোর নিয়ম (Getting Started)

### প্রয়োজনীয় সফটওয়্যার
- Node.js (ভার্সন ২২ বা এর উপরের হতে হবে)
- `pnpm` (সিস্টেমে ইন্সটল করা না থাকলে `npm install -g pnpm` ব্যবহার করে নিন)

### ইনস্টলেশন (Installation Setup)

১. প্রথমে সব প্যাকেজ (Dependencies) ইনস্টল করুন:
```bash
pnpm install
```

২. প্রজেক্ট ফোল্ডারে একটি `.env` ফাইল তৈরি করুন এবং নিচের মানগুলো বসিয়ে দিন:
```env
NEXT_PUBLIC_APP_NAME="Nurjahan Diagnostic"
NEXT_PUBLIC_BASE_URL="https://nurjahandiagnostic.com"
NEXT_PUBLIC_API_URL="http://localhost:5000"
NEXT_PUBLIC_SMS_API_URL="https://api.bdbulksms.net/api.php?json"
NODE_ENV="development"
```

৩. এরপর ডেভেলপমেন্ট সার্ভার চালু করুন:
```bash
pnpm run dev
```

ব্রাউজারে গিয়ে [http://localhost:3000](http://localhost:3000) এ ভিজিট করলেই আপনার পুরো প্রজেক্টটি দেখতে পাবেন।

---
**প্রোডাকশন বিল্ড করার জন্য (Production Build):**
```bash
pnpm run build
pnpm run start
```
